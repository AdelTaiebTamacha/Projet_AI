Recherche plus court chemin

Ecole Nationale de l'Aviation Civile
IENAC17 SITA

Presentation

This program seek for the best trajectory of a vehicule in a provided .svg circuit. It is based upon an A* algorithm with a heuristic.

Usage

From terminal run the interface executable :

./interface
